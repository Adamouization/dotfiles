# Port Forwarding in VirtualBox

#### VM Settings

#### Network

#### Adapter 1

- enable network adapter:  :white_check_mark:

- attached to: **NAT**

- cable connected:  :white_check_mark:

- port forwarding: 

```
Name: <name>.
Protocol: TCP
Host IP: 127.0.0.1
Host port: 2222
Guest IP: 10.0.2.15
Guest port: 80
```

- access on host OS in local web browser at this address: `127.0.0.1:2222/`
